# Sleep Health & Lifestyle Analysis

An exploratory analysis of sleep duration, quality, stress, and physical activity in 374 healthy adults.

## Key Findings
- Higher stress levels are associated with lower sleep quality ratings
- Participants with sleep disorders average X fewer hours of sleep than those without
- Physical activity shows a positive relationship with sleep duration
- Overweight and obese participants show wider variation in sleep duration compared to normal weight participants

## Why This Project
Sleep science is foundational to how wearables platforms like Whoop, Oura, and Garmin model readiness and recovery. As a researcher with a background in sport and exercise physiology, I built this project to demonstrate how coaches, athletes, and health professionals can explore wearable data in relation to physical activity, mood state,  and body composition

## Tools
R · tidyverse · ggplot2 · janitor

## Data Source
Laksika Tharmalingam (2023). Sleep Health and Lifestyle Dataset. 
Kaggle. https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset

## How to Run
1. Clone this repo
2. Open in RStudio
3. Run scripts/sleep-health-analysis-script.R
